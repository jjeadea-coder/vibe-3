Documentation Overview

공공직군 행정업무 슈퍼앱

팀 일정 공유, 엑셀 자동화, 민원 대응 챗봇, 공공 행정 뉴스 수집 기능을 구현하기 위한 제품 요구사항, 아키텍처, 운영 가이드를 한눈에 볼 수 있도록 정리한 문서입니다.

PRD 요약

이 앱은 공공기관 행정 담당자가 반복적으로 수행하는 일정 공유, 엑셀 자료 처리, 민원 응대 초안 작성, 행정 뉴스 모니터링을 하나의 웹 환경에서 처리하도록 돕는 내부 업무 지원 도구입니다.

TypeScript Vite React Python FastAPI uv SQLite

핵심 기능

팀원 스케쥴 관리

휴가, 근무, 출장, 교육, 외근 일정을 캘린더에서 공유하고 팀원별로 필터링합니다.

엑셀 업무 자동화

엑셀 파일을 특정 컬럼 기준으로 분리하거나 여러 파일을 하나로 병합합니다.

민원 대응 챗봇

첨부된 민원 매뉴얼을 바탕으로 대응 방향, 답변 초안, 전화 스크립트를 생성합니다.

뉴스 기사 수집

공공 행정 관련 뉴스를 매일 아침 수집하고 중복 제거, 요약, 북마크를 지원합니다.

아키텍처 요약

프론트엔드는 React 기반 기능별 모듈 구조를 사용하고, 백엔드는 FastAPI 라우터, 서비스, 저장소 계층으로 분리합니다. SQLite는 MVP와 로컬 실행에 적합한 기본 DB로 사용합니다.

API 초안

  • GET /api/schedules, POST /api/schedules
  • POST /api/excel/split, POST /api/excel/merge
  • POST /api/chatbot/manuals, POST /api/chatbot/query
  • GET /api/news, POST /api/news/collect

운영 가이드

프론트엔드

cd frontend
npm run dev

백엔드

cd backend
$env:UV_CACHE_DIR="..\.uv-cache"
uv run uvicorn app.main:app --reload

현재는 코드 파일이 아직 없으므로 실행 명령은 향후 React 진입점과 FastAPI 앱 작성 후 사용합니다.

주요 리스크

  • 민원 챗봇 답변의 행정적, 법적 정확성 검증이 필요합니다.
  • 엑셀 파일 포맷 다양성으로 인해 처리 오류가 발생할 수 있습니다.
  • 공공기관 내부망에서는 외부 뉴스 수집이 제한될 수 있습니다.
  • 민원 매뉴얼과 업로드 파일에는 개인정보가 포함될 수 있어 보안 정책이 필요합니다.
  • SQLite는 초기 MVP에는 적합하지만 동시성 요구가 커지면 DB 전환이 필요합니다.
상세 내용은 같은 폴더의 PRD.md, Architexture.md, Operation.md를 기준 문서로 사용합니다.
